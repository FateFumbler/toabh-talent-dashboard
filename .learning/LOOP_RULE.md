# Loop Prevention Rule

## What is a Loop?
Asking the same or similar clarifying question 2+ times instead of acting.

## Examples
- "What kind of modifications would you like?"
- "Can you clarify what you mean?"
- "Would you like me to change X or Y?" (after already being told to change X)

## What To Do Instead
1. Read the user's last message carefully
2. Make your best interpretation
3. Execute immediately
4. Show results, not questions

## If Uncertain
- Choose the most likely interpretation
- Do it
- User will correct if wrong
- Being wrong once is better than asking 5 times
